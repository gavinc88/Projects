Project Title: Logo Interpreter

Author: Gavin Chu

Description:
This is a simplified version of a interpreter for the programing language,
Logo, developed at UC Berkeley, which is similar to Scheme.

Instruction:
- I implemented "logo.py"
- run "logo.py" with Python 3 and the interpreter will run
- try simple commands such as
	- print "a
	- sum 1 2
	- print difference 3 1
